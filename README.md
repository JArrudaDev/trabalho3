# Dashboard Mercadinho

Este projeto cria um **dashboard completo** para um mercadinho usando Python, Dash e Plotly.

## Funcionalidades

- Gráfico de vela (Fluxo de caixa)
- Gráfico de coluna (Produtos mais vendidos)
- Gráfico de pizza (Distribuição de vendas)
- WordCloud (Produtos vendidos)
- Controle de estoque
- Controle de validade dos produtos
- Controle de fiados e top 10 clientes
- Entrada e saída de caixa

## Instalação

```bash
pip install -r requirements.txt
python app.py
